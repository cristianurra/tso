¿Cómo ejecutar el programa?

La compilación se realiza con ./make.sh

Se ejecuta con ./main.c

AL ejecutar este eliminara la memoria compartida previamente creada. Eso se hace al inicicio de la ejecución solo para permitir su revisión despues de terminado los procesos.
